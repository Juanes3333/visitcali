# Descubre Cali - Landing

Landing page para promocionar una experiencia cultural en Cali y motivar a las personas a conectar con la ciudad. El sitio presenta una propuesta de valor clara, tres pilares de contenido y un llamado a la accion hacia contacto.

## Que incluye

- Hero con mensaje principal y CTA.
- Seccion de contexto con propuesta hibrida (virtual/presencial).
- Tres tarjetas con video para destacar pilares.
- CTA final centrado hacia la pagina de contacto.
- Pagina de contacto con informacion basica y enlaces.

## Estructura del proyecto

- [index.html](index.html): landing principal.
- [contact.html](contact.html): pagina de contacto.
- [style.css](style.css): estilos globales y responsive.
- [script.js](script.js): efectos de header y videos en hover.
- [media/](media/): imagenes y videos del sitio.

## Como usarlo

Abrir [index.html](index.html) en el navegador.

Si quieres usar un servidor local:

```bash
python -m http.server 3000
```

Luego visita:

```
http://127.0.0.1:3000/index.html
```
